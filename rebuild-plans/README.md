# 블로그 리빌드 계획 보관소

블로그와 시리즈 개편 계획을 장기적으로 누적·관리하는 폴더입니다.

## 운영 원칙

- 실제 확정된 결정과 검토 중인 제안을 구분한다.
- 기존 공개 주소를 삭제하거나 바꾸기 전에 연결 구조를 먼저 점검한다.
- 블로그별 역할과 시리즈별 경계를 문서로 확정한 뒤 대문·카테고리·개별 글을 수정한다.
- 작업이 끝날 때마다 해당 계획서의 진행 상태와 변경 이력을 갱신한다.
- 임시 링크와 확인되지 않은 주소는 계획서에 넣지 않는다.
